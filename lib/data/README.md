# data
Contiene las fuentes de datos, modelos y la implementación de los repositorios.
- datasources: Clases que obtienen datos de APIs o almacenamiento local.
- models: Definición de modelos de datos (cómo se guardan y procesan).
- repositories_impl: Implementación concreta de los repositorios (cómo se conectan los datos con el dominio).
